# 世界杯每日赛程预测活动 - UI 设计规范

> 本文稿基于《营销活动 H5 通用设计规范 (V2.0 整合版)》衍生，作为「世界杯每日赛程预测」活动的专属设计执行指南。

---

## 1. 🌈 核心视觉哲学
- **高亮主题 (High-Key Style)**：世界杯是一场盛会，整体须保持**通透、明快、极具热情的竞技感调性**。严禁使用暗沉色调或单调的绝对黑。主色调应围绕赛事主题展开（如盛大、激情、荣耀）。
- **底色同步定律 (Global Background Sync)**：此为页面质感的核心红线。页面的背景画布必须精准提取 KV 头图最底边缘的色值。在滑动体验中，底色和 KV 应当完全融为一体，不可出现哪怕 1px 的断层线。

## 2. 🏗️ 头部与 Logo 规范
- **Zhibo8 品牌标识**：
 - **位置**：页面顶部正中央绝对定位（高 24~28px，距离顶部 20px）。
- **KV 主视觉规范**：
 - **宽度限定**：固定 390px 满宽。
 - **画风红线**：高阶 3D 风格建模或高品质商业插画，需突显高级感。绝对禁止出现低幼或卡通风格。
 - **主标题字体**：应用具备质型体积感（3D挤压或光效描边）的中文字体，确保单行舒展居中。如果头图背景偏复杂，需在标题下层叠加 10%~15% 的白色或环境色径向弥散遮罩（Scrim）增强对比度，**严禁用 #000000 描边**。

## 3. 🌊 页面空间与层叠布局
- **负边距搭叠结构**：
 - 核心模块【个人战绩摘要条】和【预测卡片】应当有强烈的景深感。
 - 最顶部的容器卡片（如战绩摘要/或当天的首张赛程卡）须配置 `-40px 至 -60px` 的负边距，搭叠在 KV 下方，并采用大圆角设计（建议 `border-radius: 28px` 或以上）。
- **空间碎片氛围支持**：背景空隙可适当飘落符合赛事主题的微粒（不超过 3 个元素，透明度限定于 5%-10% 内层），构建“活体环境感”。

## 4. 💎 控件与面板的"纳米级"触感设计
### 4.1 内容面板（战绩、达人榜与赛程卡）
- **面板材质**：采用轻盈通透的微渐变背景色或高品质玻璃拟态（Glassmorphism），边界干净利落。
- **排版修正**：各卡片内外边距需呼吸感充足。文字需要严格排版避头尾，标题保持紧凑间距 `-0.5px`，辅助文字色值确保无障碍阅读，明亮底色下严禁浅于 `#94A3B8`（推荐以 `#64748B` 为底线）。

### 4.2 互动控件（预测按钮、分享按钮）
所有的可点击预测选项按钮（如胜平负、进球数、达人榜Tab、分享战绩）禁止大色块生硬扁平设计。
- **渐变感**：按钮须应用流顺的微渐变（320°方向），色阶起伏控制在 5%。
- **触感装甲系统**：
 1. 顶部勾勒内发光轮廓线：`box-shadow: inset 0 1px 0 rgba(255,255,255,0.4)`。
 2. 极具呼吸感的弥散阴影：Blur模糊半径 ≥ 30px，Opacity 透明度 ≤ 30%。
- **交互状态区分**：
 - 【未选中选项】：细腻描边，主打轻柔克制的外观。
 - 【已选中选项 / 点击态】：填充主题高亮色及内阴影变化，并展现实时投票百分比条。

## 5. ⚖️ 底部抗真空防虚脱策略
针对单日赛程较少（比如淘汰赛只有 1 场），可能无法占满屏幕高度的场景：
- **物理沉淀感底座**：不可在活动页尾出现超过 30vh 的白屏底。需要通过在页面最下端叠加较深的同色系环境渐变位，将“悬浮状态”转变为坚固的“重力压舱感”。
- **社交背书撑场**：如果在底部有空虚感，请拉伸“活动规则”或增加实时社交氛围文字（例如：`今日已有 23,456 人参与疯狂预测`）并吸附于最后侧，确保维持85%以上的全屏内容覆盖张力。
