# 世界杯每日赛程预测活动 UI 设计规范（用户提供原文归档）

> 来源：Feishu 群聊用户粘贴正文
> 说明：作为后续设计阶段的输入约束，不视为最终设计交付物

## 已确认设计约束
- 页面形态：营销活动 H5 / mobile-web
- 视觉调性：高亮、通透、竞技感强，禁止暗沉或低幼卡通风格
- KV 约束：390px 满宽，底色需与页面背景无缝衔接
- 层叠结构：战绩摘要或首卡需以 -40px 至 -60px 负边距搭叠 KV
- 卡片材质：轻盈微渐变 / 玻璃拟态
- 按钮质感：微渐变、内发光轮廓、弥散阴影，避免纯扁平大色块
- 底部策略：低赛程日不得出现大面积空白，需通过环境渐变或扩展内容撑满页面

## 备注
该文档是设计输入，不等同于 design.md + 高保真 mockups，正式进入 design 阶段后仍需产出可评审设计稿。
