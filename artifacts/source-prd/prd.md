## 1. 活动概况

### 1.1 实现目标

| 目标维度 | 具体指标 |
|---------|---------|
| **拉活 / 促留存** | 通过每日预测的打卡式参与感，提升世界杯期间 DAU 及次日留存率 |
| **导流至比赛内页** | 从活动页引导用户跳转至直播吧比赛综合内页，拉动比赛内页 UV 及停留时长 |

### 1.2 活动内容

世界杯期间，每日根据当日赛程为用户提供比赛预测玩法。用户可对每场比赛的 **胜平负**、**全场进球数**、**指定球员能否进球** 三道预测题进行投票，赛后自动揭晓结果。预测后可查看全站投票分布比例，形成"民间赔率"的群体博弈感。设有预测达人榜 TOP20 和个人战绩分享卡，增强社交竞争感和传播力。无任何实物/虚拟奖励，纯粹依靠预测命中的成就感驱动参与。

### 1.3 活动时间

- **活动周期**：2026 世界杯赛程期间（具体日期待定）
- **每日预测截止时间**：每场比赛开赛即截止该场预测

### 1.4 适配关键词

- 世界杯预测
- 世界杯竞猜
- 赛程预测

### 1.5 适配要求

本活动需适配 **iOS**、**Android** 以及 **鸿蒙 OS (HarmonyOS)** 设备。

## 3. 页面结构与交互设计

### 3.1 页面信息架构（IA）

```
活动首页
├── 顶部 KV / 活动标题区
├── 个人战绩摘要条
│ ├── 用户头像（居左极简展示）
│ ├── 总预测场次
│ ├── 总命中率
│ └── 「分享战绩」按钮
├── 右上角悬浮入口（固定呈现）
│ ├── 活动规则按钮（点击打开规则弹窗）
│ └── 预测达人榜按钮（点击进入达人榜详情）
├── 日期选择器（可左右滑动切换比赛日，默认定位当日）
└── 当日赛程预测卡片列表
 └── 比赛卡片（可多张，按开赛时间排序）
 ├── 对阵信息区（队徽 + 队名 + 开赛时间 + 赛事轮次）
 ├── 预测题 ❶ 胜平负
 ├── 预测题 ❷ 全场进球数
 ├── 预测题 ❸ 指定球员能否进球
 └── 「提交预测」交互按钮

预测达人榜页面（从首页入口进入）
├── 榜单切换 Tab：「每日榜」 / 「总榜」
├── 排行榜列表（TOP20）
│ └── 每行：排名 + 用户头像 + 昵称 + 命中题数
├── 我的排名条（固定底部）
│ └── 当前用户排名 + 头像 + 昵称 + 命中题数
└── 返回首页
```

### 3.2 比赛卡片详细设计

每张比赛卡片包含以下内容：

#### 3.2.1 对阵信息区

- 主队队徽 + 主队名称
- "VS" 分隔
- 客队队徽 + 客队名称
- 赛事轮次标签（如：小组赛A组 / 1/8决赛）
- 开赛时间（北京时间）
- **交互**：点击对阵双方头像或比赛时间区域，即可跳转至直播吧该场比赛的综合内页

#### 3.2.2 预测题区域

**❶ 比赛结果（胜平负）**

| 选项 | 说明 |
|------|------|
| 主胜 | 主队获胜 |
| 平 | 双方打平 |
| 客胜 | 客队获胜 |

**❷ 全场进球数**

| 选项 | 说明 |
|------|------|
| 0-1球 | 全场总进球为0或1个 |
| 2-3球 | 全场总进球为2或3个 |
| 4球以上 | 全场总进球≥4个 |

**❸ 指定球员能否进球（主队 VS 客队各一名球员）**

展示格式为主队球员 VS 客队球员的对比形式，每位球员各自独立预测：

| 球员位置 | 选项 | 说明 |
|---------|------|------|
| 主队球员 | 能 / 不能 | 该球员在本场比赛中是否进球 |
| 客队球员 | 能 / 不能 | 该球员在本场比赛中是否进球 |

- 球员信息由编辑每日提供给开发进行配置维护，每场比赛提供**主队、客队各 1 名焦点球员**
- 球员展示格式：球员头像 + 球员姓名 + 所属球队，左右对比排列（主队球员居左，客队球员居右）

#### 3.2.3 提交预测按钮

- 文案：「提交预测」
- 预测前置条件：必须同时完成比赛结果、全场总进球数、指定焦点球员能否进球（3道题全部完成）才可提交。
- 点击交互逻辑：
 - **预测截止（超过预测时间）**：若比赛进行中或已结束，点击弹出 Toast 提示「超过预测时间」。
 - **信息未填全**：未答完所有题目时点击，弹出 Toast 提示「请完成本场所有预测」。
 - **校验通过**：提交成功，更新为已预测完成态。

### 3.3 核心交互流程

#### 3.3.1 预测流程

```
用户进入活动页
 → 查看当日赛程列表
 → 选择感兴趣的比赛（可选择性参与，非强制全部预测）
 → 点击三道预测题的选项（必须全部选择）
 → 点击「提交预测」按钮
 → 返回成功，选项均变为选中态高亮（不可取消、不可修改）
 → 并展示各题的全站投票分布百分比
 → 开赛后该场所有预测题锁定
 → 赛后自动结算，揭晓预测结果
```

#### 3.3.2 预测状态说明

- **预测前**：选项为可点击态，不显示投票分布
- **预测后**：选中选项高亮，**显示全站投票分布百分比**，不可再修改
- **已锁定**：开赛后未预测的用户显示"预测已截止"，已预测的用户显示"等待开奖"
- **已结算**：显示 ✅ / ❌ 标记 + "你击败了 X% 的预测者"
- **已取消**：特殊情况（比赛推迟/球员未出场）显示"预测已取消"

### 3.4 个人战绩摘要

位于页面顶部 KV 下方，以摘要条形式展示：

| 字段 | 说明 |
|------|------|
| 用户头像 | 在条目最左侧展示当前登录用户的头像，增强个人属性 |
| 总预测场次 | 用户参与预测的比赛场次总数（不含已取消的） |
| 总命中率 | 命中题数 / **已结束并结算的**总预测题数 × 100%。【注】：未结束/未结算的比赛不计入命中率计算 |
| 「分享战绩」按钮 | 点击生成个人战绩长图，可分享至社交平台 |

- 首次进入且未参与过预测时，显示默认态：`0 场 | 命中率 --`，分享按钮仍可点击

### 3.5 日期选择器

- 支持左右滑动切换比赛日
- 日期标签仅展示简单的纯文本日期（如：6月20日），不添加诸如“今日核心战”等额外修饰文案
- 进入页面时默认定位并选中当日
- 可查看过去日期的已结算预测结果
- 未来有赛程的日期可提前预测（如果比赛信息已配置）
- 无赛事的日期不可选，显示"今日无比赛"

### 3.6 预测达人榜 TOP20

#### 3.6.1 入口设计

放弃原有的横向缩略卡，改为采用**页面右上角悬浮小按钮**设计。
与「活动规则」按钮同处于右侧悬浮区（活动规则按钮在上方，达人榜按钮在下方）。

- 按钮样式：小巧的吸边悬浮图标
- 触发动作：点击后即打开“预测达人榜”完整页面
- 展示内容说明：打开达人榜页面后，展示排行的用户排名、头像、名称、猜对的题目数

#### 3.6.2 达人榜完整页面

包含顶部 Tab 切换（默认选中「每日榜」）：

- **每日榜**：展示单日内比赛的命中题数排行（可横滑切换查看往期比赛日）
- **总榜**：展示整个活动期间累计的命中题数排行

| 元素 | 说明 |
|------|------|
| 排行列表 | 依据当前选中的榜单展示 TOP20 用户，每行包含：排名序号、用户头像、用户昵称、命中题数 |
| 前三名特殊样式 | 第 1/2/3 名使用金/银/铜色奖牌样式区分 |
| 我的排名（底部固定） | 无论当前用户是否进入 TOP20，页面底部始终固定展示"我的排名"条，包含：当前对应榜单排名、头像、昵称、命中题数 |

#### 3.6.3 排名规则

- **排名维度**：命中题数（猜对的预测题总数），降序排列
- **同分处理**：命中题数相同时，按命中率高低排序；命中率仍相同时，按首次参与预测时间先后排序（先参与的排前面）
- **更新频率**：每场比赛结算后实时更新排行榜
- **已取消的题目不计入**命中题数统计

### 3.7 个人战绩分享卡

#### 3.7.1 触发入口

- 个人战绩摘要条中的 「分享战绩」 按钮

#### 3.7.2 分享卡内容

生成一张竖版长图，包含以下信息：

| 区域 | 内容 |
|------|------|
| 顶部 | 活动主题标题 + 世界杯视觉元素 |
| 用户信息 | 用户头像 + 昵称 |
| 核心数据 | 总预测场次、命中题数、总命中率 |
| 排名信息 | "超越了 X% 的预测玩家"（基于命中题数的全站排名百分位） |
| 底部 | 直播吧品牌 logo + 活动二维码（引导扫码参与） |

#### 3.7.3 交互流程

```
用户点击「分享战绩」
 → 前端 Canvas 生成战绩长图
 → 弹出分享面板 / 保存到相册

```

| 场景 | 处理方式 |

## 4. 活动规则

### 4.1 参与门槛

- **需登录**：未登录用户可浏览活动页面和赛程信息，点击预测选项时唤起登录弹窗
- **无其他门槛**：所有已登录用户均可参与

### 4.2 预测规则

1. 用户可自由选择参与预测的比赛场次，不要求全部参与
2. 每道预测题独立参与，可只预测其中1题或2题
3. **预测一经提交不可修改**
4. 预测截止时间为该场比赛开赛时刻。**如果比赛已开始、进行中或已结束，则用户无法再进行预测**。
5. 本活动**无任何奖品奖励**，为纯预测玩法

### 4.3 结算规则

| 项目 | 规则 |
|------|------|
| **胜平负** | 以**常规90分钟**比赛结果为准。淘汰赛如打至加时赛/点球大战，常规时间平局则胜平负判定为"平" |
| **全场进球数** | 以**常规90分钟**进球总数为准，加时赛进球不计入 |
| **球员进球** | 以国际足联官方赛后统计为准。**乌龙球不计为该球员进球** |
| **击败百分比** | 该题选中正确选项的用户占比，用"你击败了 X% 的预测者"展示（X = 选中错误选项的用户占比） |

### 4.4 达人榜规则

| 项目 | 规则 |
|------|------|
| **榜单类型** | 分为「每日榜」与「总榜」。每日榜统计用户在具体某一比赛日的命中题数；总榜统计活动累计命中题数 |
| **排名维度** | 按对应榜单的累计命中题数降序排列 |
| **同分处理** | 命中题数相同时，按在该榜单时间范围内的命中率高低排序；仍相同时，按首次参与预测时间先后排序 |
| **更新频率** | 当日或相关往期比赛结束并结算后，排行榜实时更新 |
| **已取消题目** | 不计入命中题数统计 |

### 4.5 异常场景处理

| 异常场景 | 处理方式 | 用户端展示 |
|---------|---------|-----------|
| **比赛推迟** | 该场全部预测题作废，不计入个人战绩统计 | 卡片显示："比赛推迟，预测已取消" |
| **比赛取消** | 该场全部预测题作废，不计入个人战绩统计 | 卡片显示："比赛取消，预测已取消" |
| **指定球员赛前伤退 / 未出场** | 仅该球员进球题作废，胜平负和进球数题正常结算 | 球员进球题显示："球员未出场，预测已取消" |
| **数据延迟** | 允许赛后延迟结算（最长24小时内完成） | 卡片显示结算中loading态："赛果确认中" |
| **未登录点击预测** | 弹出登录弹窗 | 标准登录流程 |
| **当日无赛事** | 日期选择器该日不可选 | 显示"今日无比赛" |

## 5. 比赛卡片状态生命周期

```
┌──────────┐ 开赛哨响 ┌──────────┐ 赛后结算 ┌──────────┐
│ 可预测 │ ──────────────→ │ 已锁定 │ ──────────────→ │ 已结算 │
└──────────┘ └──────────┘ └──────────┘
 │ │
 │ 比赛推迟/取消/球员未出场 │
 └─────────────────→ ┌──────────┐ ←────────────────────────┘
 │ 已取消 │ (极端情况赛后判定取消)
 └──────────┘
```

| 状态 | 用户交互 | 展示内容 |
|------|---------|---------|
| **可预测** | 可点击选项提交预测 | 选项按钮可点击态，未显示投票分布（预测后显示） |
| **已锁定** | 不可交互 | 已预测用户："等待开奖" + 已选选项高亮 + 投票分布；未预测用户："预测已截止" |
| **已结算** | 不可交互 | ✅/❌ 标记 + 正确答案高亮 + 投票分布 + "击败了X%的预测者" |
| **已取消** | 不可交互 | 相应取消文案提示 |

## 6. 数据配置说明

### 6.1 赛程数据

- 赛程数据来源：直播吧现有赛事数据源
- 需要的数据字段：比赛 ID、主队信息（队名+队徽）、客队信息（队名+队徽）、开赛时间、赛事轮次、比赛状态、最终比分

### 6.2 球员进球题配置

- **配置方式**：编辑每日提供球员名单 → 开发人员手动配置
- **配置时间**：需在该场比赛开赛前完成配置
- **配置内容**：每场比赛主队、客队**各 1 名**焦点球员，包含球员姓名、球员头像、所属球队
- **异常兜底**：如编辑未及时提供某场比赛的球员信息，该场仅展示胜平负和进球数两道题；如只提供了一方球员，仅展示该方球员的预测题

## 7. 数据埋点

### 📈 业务分析与核心指标

- **核心路径**：用户进入活动页 → 浏览赛程 → 参与预测 → 查看比赛详情（导流）→ 赛后查看预测结果 → 分享战绩
- **核心指标**：
 - 活动页 PV/UV
 - 预测参与率（参与预测的用户数 / 活动页 UV）
 - 单题预测参与率（各题型的参与人数对比）
 - 比赛详情跳转率（点击跳转 / 活动页 UV）
 - 次日回访率（次日再次进入活动页的用户占比）
 - 达人榜页面 UV
 - 分享卡生成次数 / 分享成功次数

## 9. 视觉风格建议

- **整体风格**：世界杯主题，热烈、大气、竞技感强
- **卡片设计**：每张比赛卡片使用圆角容器，内含对阵信息和预测题，层次分明
- **预测选项**：
 - 未选中态：描边按钮，简洁轻量
 - 选中态：填充高亮（主题色），并展示投票分布的进度条
 - 已结算-命中态：绿色系 ✅ 标记
 - 已结算-未命中态：灰色/红色系 ❌ 标记
- **投票分布**：每个选项后附带百分比进度条，直观展示全站投票比例
- **队徽展示**：使用标准队徽素材，关注展示清晰度
- **达人榜**：前三名使用金/银/铜奖牌图标突出展示，列表行间距适中保证可读性
- **分享卡**：世界杯主题背景 + 数据可视化排版，品牌感强，适合社交传播

## 附录：活动规则文案（用户端展示）

**活动规则**

1. 世界杯比赛期间，每日更新当日赛程预测题目
2. 用户可自由选择感兴趣的比赛进行预测，预测一经提交不可修改
3. 预测截止时间为每场比赛开赛时刻
4. 每场比赛设有三道预测题：比赛结果（胜平负）、全场进球数、指定球员能否进球
5. 预测后可查看全站球迷的投票分布
6. 比赛结果以常规90分钟为准（加时赛及点球大战不计入）
7. 球员进球以国际足联官方统计为准，乌龙球不计为进球
8. 如比赛推迟或取消，相关预测自动作废
9. 预测达人榜分为总榜和每日榜，按对应时期的命中题数排名，赛后实时更新
10. 本活动为纯预测玩法，不涉及任何奖品奖励
11. 本活动最终解释权归直播吧所有
