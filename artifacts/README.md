# Artifacts

Use this folder for the human-readable and machine-routable delivery artifacts.

Recommended files:

- `idea.md`
- `proposal.md`
- `specs/`
- `design.md`
- `tasks.md`
- `client-implementation.md`
- `backend-implementation.md`
- `test-report.md`
- `release-plan.md`

## Required conventions

- `proposal.md` explains why the change exists and what changes.
- `specs/` contains requirements, scenarios, and acceptance criteria.
- `design.md` includes React Native high-fidelity mockup references, page inventory, state coverage, and implementation constraints.
- `tasks.md` is the only approved implementation checklist for `/opsx:apply`.
- Implementation and test reports must reference the approved specs and design baseline.
